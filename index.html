ANS -

Web1 refers to the early days of the internet, where web pages were static and read-only, with minimal interactivity.
Web2 is the current state of the internet, where users can interact with web pages and generate user-generated content. Social media platforms, blogs, wikis, and online forums are examples of Web2 technologies.
Web3 is the next evolution of the internet, with a focus on decentralization, security, and privacy. Decentralized applications (dApps) built on blockchain technology are examples of Web3 technologies.
When searching for the side effects of medicines, a user on Web1 would typically encounter a static webpage with plain text containing information about the medicine and its side effects. On the other hand, a user on Web3 would have access to a decentralized application built on blockchain technology, where the data is verified, secure, and up-to-date. The user can interact with the application by leaving comments, contributing information, and sharing their experiences, leading to more reliable and trustworthy information. Additionally, the user has control over their data and can decide how it is used and shared.
